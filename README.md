# Callcenter CRM (Laravel)

Это стартовый проект Laravel под интерфейс как на ваших скринах: **сделки (канбан/список/карточка)** + **задачи/дела** + **лента активности** + **чаты** + **источники** + **кастомные поля**.

## Быстрый старт

1. Установите зависимости:
   ```bash
   composer install
   ```
2. Скопируйте env:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```
3. Настройте MySQL в `.env` (DB_*).
4. Миграции + сиды:
   ```bash
   php artisan migrate --seed
   ```
5. Запуск:
   ```bash
   php artisan serve
   ```

### Доступ по умолчанию (после seed)
- Email: `admin@example.com`
- Password: `password`

## Что есть сейчас
- Авторизация (простая форма логина на сессиях)
- Страницы:
  - `/deals/kanban` — канбан
  - `/deals` — список
  - `/deals/create` — создание лида вручную
  - `/deals/{id}` — карточка сделки + задачи + лента
- База данных: таблицы под функционал со скринов (см. `database/migrations`)

Дальше можно добавлять интеграцию бота (webhook) — но это второй этап.
