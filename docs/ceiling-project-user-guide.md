# Руководство по проектировке потолков

## 1. Как открыть проектировку

1. Откройте верхнее меню `Проектировка`.
2. Создайте новый проект.
3. При необходимости привяжите проект к сделке.
4. Загрузите фото замера или чертеж от замерщика.

## 2. Как открыть чертеж комнаты

1. Добавьте комнату в проект.
2. В списке комнат нажмите `Открыть чертеж комнаты`.
3. Страница прокрутится к блоку `Чертеж / Canvas комнаты`.

Если комната уже выбрана, можно нажать большую кнопку `Открыть чертеж` в верхней части проекта.

## 3. Основные режимы чертежа

### Точка / угол

Используйте этот режим, чтобы собирать контур комнаты.

- Клик по схеме добавляет новую точку в ближайшее ребро.
- Черные точки можно тянуть мышкой.
- Колесо мыши приближает и отдаляет схему.
- Пробел + перетаскивание или средняя кнопка мыши двигают поле чертежа.
- Двойной клик по точке удаляет ее.

### Сдвиг стены

Используйте этот режим, если нужно двигать не одну точку, а целую стену.

- Кликните по нужной стене.
- Выбранная стена подсветится.
- Потяните ее мышкой.

### Поставить элемент

Используйте этот режим для света, труб, ниш, карнизов и других элементов.

- Для свободного элемента клик по схеме подставляет `X/Y`.
- Для ниши или карниза выберите размещение `По стене`.
- После этого кликните по нужной стене.

## 4. Что означают кнопки

### Разрезать стену

Вставляет новую точку посередине выбранной стены.

Используйте, когда нужно:

- разбить длинную стену на две;
- сделать выступ;
- подготовить сложную форму комнаты.

### Ортоснап

Помогает держать точки и стены ближе к вертикали и горизонтали.

Если рисование идет неровно, оставьте его включенным.

### Сбросить в прямоугольник

Возвращает комнату к базовой прямоугольной форме по текущим размерам.

## 5. Как нарисовать комнату с нуля

1. Добавьте комнату.
2. Откройте чертеж комнаты.
3. Если есть фото, используйте его как подложку.
4. В режиме `Точка / угол` обведите контур.
5. При необходимости передвиньте стены в режиме `Сдвиг стены`.
6. Нажмите `Сохранить геометрию`.

## 6. Как добавить элементы

### Спот, люстра, труба, вентиляция

1. В блоке `Элементы комнаты` выберите тип.
2. Оставьте размещение `Свободно`.
3. Нажмите `Взять координату со схемы` или просто переключитесь в режим `Поставить элемент`.
4. Кликните по нужному месту на схеме.
5. Сохраните элемент.

### Ниша или карниз

1. Выберите тип `Ниша` или `Карниз`.
2. Поставьте размещение `По стене`.
3. Кликните по нужной стене на схеме.
4. Проверьте номер стены, смещение и длину.
5. Сохраните элемент.

## 7. Как читать чертеж

- Синие линии показывают контур комнаты.
- Красным выделяется выбранная стена.
- Подпись `#1`, `#2` и т.д. показывает номер стены.
- Рядом с номером показывается длина стены в сантиметрах.
- Цветные маркеры показывают элементы комнаты.

## 8. Что обязательно сохранять

В проектировке есть два типа сохранения:

- `Сохранить геометрию` сохраняет контур комнаты;
- `Сохранить` у конкретного элемента сохраняет его положение и параметры.

Если двигали стены или углы, нажмите `Сохранить геометрию`.

Если двигали нишу, карниз, светильник или другой элемент, сохраните сам элемент.

## 9. Частые ошибки

### Не видно канвас

- Проверьте, что выбрана хотя бы одна комната.
- Нажмите `Открыть чертеж комнаты`.

### Элемент не ставится на стену

- Убедитесь, что в форме элемента выбрано размещение `По стене`.
- Потом кликните именно по линии стены, а не по пустому месту.

### Двигается слишком резко

- Оставьте `Ортоснап` включенным.
- Для точной правки используйте поля координат справа, они работают в сантиметрах.
- Клик по строке точки справа выбирает нужный угол и подсвечивает его на схеме.

### Изменения пропали

- После контура нажимайте `Сохранить геометрию`.
- После элемента нажимайте `Сохранить` у этого элемента.

## 10. Рекомендуемый порядок работы

1. Сделка или standalone-проект.
2. Фото замера.
3. Комнаты.
4. Геометрия комнаты.
5. Ниши, карнизы, свет, трубы.
6. Проверка размеров стен.
7. Сохранение.
8. Проверка сметы.
